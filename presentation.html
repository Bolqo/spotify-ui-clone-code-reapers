Process

Git Workflow

Repository Setup
Irakli creates repo, builds folder structure, sets up main.scss entry point and all SCSS partials
Feature Branches
Each team member works on their own branch — no direct commits to main
Pull Requests & Review
Code reviewed before merging — consistent naming, no !important, proper @use imports
Deadline: March 13, 2026
All 16 screens delivered and merged to main on time
# branch per feature
git checkout -b feat/player-bar

# compile SCSS
npm run sass:watch

# commit with clear message
git commit -m "add player bar styles"

# open PR → review → merge
git push origin feat/player-bar
